---

Oracle

```SQL
SELECT
    FLOOR(PRICE / 10000) * 10000 PRICE_GROUP,
    COUNT(*) PRODUCTS
FROM
    PRODUCT
GROUP BY
    FLOOR(PRICE / 10000) * 10000
ORDER BY
    PRICE_GROUP

```

---

MySQL

```SQL
SELECT
    FLOOR(PRICE / 10000) * 10000 PRICE_GROUP,
    COUNT(*) PRODUCTS
FROM
    PRODUCT
GROUP BY
    PRICE_GROUP
ORDER BY
    PRICE_GROUP

```

---

MySql -> GROUP BY, HAVING, ORDER BY -> 별칭 사용 가능
Oracle -> ORDER BY만 별칭 사용 가능
