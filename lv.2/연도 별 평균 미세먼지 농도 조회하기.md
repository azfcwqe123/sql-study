MySQL

```SQL
SELECT 
    YEAR(YM) YEAR,
    ROUND(AVG(PM_VAL1), 2) "PM10",
    ROUND(AVG(PM_VAL2), 2) "PM2.5"
FROM 
    AIR_POLLUTION
WHERE 
    LOCATION2 = '수원'
GROUP BY
    YEAR(YM)
ORDER BY
    YEAR
```

---

| 함수        | 설명                       | 예시                          |
| --------- | ------------------------ | --------------------------- |
| `COUNT()` | 그룹별 행의 개수 반환 (`NULL` 제외) | `COUNT(*)`, `COUNT(column)` |
| `SUM()`   | 그룹별 합계 계산                | `SUM(sales)`                |
| `AVG()`   | 그룹별 평균 계산                | `AVG(score)`                |
| `MIN()`   | 그룹별 최소값 반환               | `MIN(price)`                |
| `MAX()`   | 그룹별 최대값 반환               | `MAX(price)`                |
