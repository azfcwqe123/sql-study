---

Oracle

```SQL
SELECT 
    COUNT(*) AS count
FROM (
    SELECT 
        COUNT(NAME) 
    FROM 
        ANIMAL_INS 
    WHERE 
        NAME IS NOT NULL 
    GROUP BY 
        NAME
)
```

---

MySQL

```SQL
SELECT COUNT(DISTINCT NAME) #NULL은 집계에서 제외됨
FROM ANIMAL_INS
```
