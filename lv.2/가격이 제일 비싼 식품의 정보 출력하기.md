Oracle

방법 1
```SQL
SELECT *
FROM (
    SELECT *
    FROM FOOD_PRODUCT
    ORDER BY PRICE DESC
)
WHERE ROWNUM = 1
```

&nbsp;

방법2
```sql
SELECT *
FROM FOOD_PRODUCT
WHERE 
    PRICE = (SELECT MAX(PRICE) FROM FOOD_PRODUCT)
```

---

MySQL

```SQL
SELECT *
FROM FOOD_PRODUCT
ORDER BY PRICE DESC
LIMIT 1
```
