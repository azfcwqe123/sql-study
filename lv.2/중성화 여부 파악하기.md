Oracle

```SQL
SELECT
    ANIMAL_ID,
    NAME,
    CASE 
        WHEN SEX_UPON_INTAKE LIKE 'Spayed%' 
            OR SEX_UPON_INTAKE LIKE 'Neutered%' THEN 'O'
        ELSE 'X'
    END AS 중성화
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```

---

MySQL

풀이1
```SQL
SELECT
    ANIMAL_ID,
    NAME,
    CASE
        WHEN (SEX_UPON_INTAKE LIKE 'Neutered%') OR (SEX_UPON_INTAKE LIKE 'Spayed%') THEN 'O'
        ELSE 'X'
    END 중성화
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```

풀이2
```SQL
SELECT
    ANIMAL_ID,
    NAME,
    IF(SEX_UPON_INTAKE LIKE 'Neutered%' OR SEX_UPON_INTAKE LIKE 'Spayed%', 'O', 'X') AS 중성화
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```

풀이3 (정규 표현식)
```SQL
SELECT
    ANIMAL_ID,
    NAME,
    IF(SEX_UPON_INTAKE REGEXP '^(Neutered|Spayed)', 'O', 'X') AS 중성화
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```

^ : 문자열 시작
