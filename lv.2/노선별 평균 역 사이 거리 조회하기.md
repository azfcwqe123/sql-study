MySQL

```SQL
SELECT 
    ROUTE,
    CONCAT(ROUND(SUM(D_BETWEEN_DIST), 1), 'km') TOTAL_DISTANCE,  
    CONCAT(ROUND(AVG(D_BETWEEN_DIST), 2), 'km') AVERAGE_DISTANCE
FROM
    SUBWAY_DISTANCE
GROUP BY
    ROUTE
ORDER BY ROUND(SUM(D_BETWEEN_DIST), 1) DESC # 문자열로 정렬하지말고 숫자로 해야함.
```
