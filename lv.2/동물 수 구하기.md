Oracle

```SQL
SELECT COUNT(*)
FROM ANIMAL_INS
```

---

MySQL

```SQL
SELECT COUNT(*)
FROM ANIMAL_INS
```

---

### HINT

COUNT()는 NULL을 제외하고 집계함. 

COUNT(NAME) 같은거로 해버릴시, NULL은 집계를 해버리지 않기 때문에 문제의 의도와 다르게 답이 나올 수 있음.
