Oracle

```SQL
SELECT
    B.INGREDIENT_TYPE, SUM(TOTAL_ORDER) TOTAL_ORDER
FROM
    FIRST_HALF A
INNER JOIN
    ICECREAM_INFO B
    ON A.FLAVOR = B.FLAVOR
GROUP BY 
    B.INGREDIENT_TYPE
ORDER BY 
    TOTAL_ORDER

```
