---

Oracle

```SQL
SELECT 
    A.BOOK_ID, 
    B.AUTHOR_NAME, 
    TO_CHAR(A.PUBLISHED_DATE, 'YYYY-MM-DD') PUBLISHED_DATE
FROM 
    BOOK A 
INNER JOIN 
    AUTHOR B
    ON A.AUTHOR_ID = B.AUTHOR_ID
WHERE
    CATEGORY = '경제'
ORDER BY 3
```

---

MySQL

```SQL
SELECT 
    A.BOOK_ID, 
    B.AUTHOR_NAME, 
    DATE_FORMAT(A.PUBLISHED_DATE, '%Y-%m-%d') PUBLISHED_DATE
FROM 
    BOOK A 
INNER JOIN 
    AUTHOR B
    ON A.AUTHOR_ID = B.AUTHOR_ID
WHERE
    CATEGORY = '경제'
ORDER BY 3
```
