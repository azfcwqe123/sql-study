Oracle

풀이1
```SQL
SELECT MAX(DATETIME) AS 시간
FROM ANIMAL_INS
```
&nbsp;

풀이2
```SQL
SELECT DATETIME AS 시간
FROM 
(
    SELECT DATETIME 
    FROM ANIMAL_INS
    ORDER BY DATETIME DESC
)
WHERE ROWNUM <= 1
```
