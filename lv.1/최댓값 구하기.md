---

Oracle

풀이1
```SQL
SELECT MAX(DATETIME) AS "시간"
FROM ANIMAL_INS
```
&nbsp;

풀이2
```SQL
SELECT DATETIME AS "시간"
FROM 
(
    SELECT DATETIME 
    FROM ANIMAL_INS
    ORDER BY DATETIME DESC
)
WHERE ROWNUM <= 1
```


---

MySQL

```SQL
SELECT MAX(DATETIME) AS "시간"
FROM ANIMAL_INS
```

### 참고

' '은 sql에서만 가능하고, 오라클에서는 안 됨.
" "은 sql, 오라클 둘 다 실행 가능. " "으로 통일하는게 편함
